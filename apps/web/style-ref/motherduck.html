Built on Modern Architecture

MotherDuck is built from the ground up with modern cloud architecture in mind. Our platform leverages the power of DuckDB, an in-process SQL OLAP database management system, combined with cloud storage and compute separation for ultimate flexibility and performance.

The LayoutGrid system you're experiencing right now provides a sophisticated approach to web layouts, combining the flexibility of CSS Grid with the precision of subgrid support. This allows for seamless nesting and perfect alignment across all levels of your layout hierarchy.

With automatic content width management and a responsive 12-column grid, content always looks perfect regardless of viewport size. The system handles complex layouts with ease, from simple landing pages to intricate application interfaces.

How It Works

The LayoutGrid system uses CSS variables to define a 12-column grid with automatic content width calculation. Each column's width is derived from the content width divided by the number of columns, ensuring perfect proportions at any screen size.

The main grid container creates three columns: a left margin, the content area (subdivided into 12 columns), and a right margin. This ensures your content is always centered and properly bounded, providing a professional and polished appearance.

Subgrid support allows nested elements to align with the parent grid, creating a cohesive visual structure throughout your entire page. Elements can span multiple columns or use the full content width as needed, giving you complete control over your layout.
